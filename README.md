# DCart

DCart is a modern online shopping web application built with Next.js, Redux Toolkit, Sanity Studio, Tailwind CSS, Firebase, NextAuth, and Stripe.

## Features

- Modern, responsive user interface
- Product catalog powered by Sanity.io
- Fully functional shopping cart (add, increase, decrease, remove, reset)
- Persistent cart state using Redux Persist
- User authentication via NextAuth / Firebase
- Safe payment flow with Stripe
- Admin dashboard powered by Sanity Studio

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
